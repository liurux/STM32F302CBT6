# STM32F302CBTx

#### Description
project about stm32f302cbt6 applying in joystick control robot and handle data receive from pc and robot

#### Software Architecture
STM32CubeMX v5.2
Keil5 MDK

#### Pin Description

1. ADC
	ADC1_IN1  PA0
	ADC1_IN2  PA1
	ADC1_IN3  PA2
	ADC1_IN4  PA3
	
	ADC2_IN1  PA4
	ADC2_IN2  PA5
	ADC2_IN3  PA6
	ADC2_IN4  PA7
	
2. USART
	USART1_TX PB3
	USART1_RX PA15
	
	USART2_TX PA9
	USART2_RX PA10
	
	USART3_TX PB10
	USART3_RX PB11
	
3. LED
	PB0
	PB1
